# 🧪 Projet QA – Test exploratoire & UX (SauceDemo)

## 🎯 Contexte

Ce projet consiste en une simulation de mission QA basée sur un site réel de démonstration e-commerce : https://www.saucedemo.com

L’objectif est de réaliser des tests exploratoires afin d’identifier des anomalies, incohérences et problèmes d’expérience utilisateur.

---

## 🧠 Méthodologie

- Test exploratoire
- Analyse UX (expérience utilisateur)
- Approche basée sur l’observation et l’investigation

---

## 🧪 Périmètre

- Connexion utilisateur
- Navigation produits
- Ajout au panier
- Processus de commande

---

## 📋 Livrables

- Charter de test exploratoire
- Notes d’exploration
- Rapport de bugs
- Captures d’écran

---

## 🛠️ Outils

- Navigateur web
- GitHub
- Documentation Markdown

---

## 📊 Résultat

Plusieurs anomalies UX et comportements inattendus ont été identifiés, notamment sur la gestion des erreurs et la cohérence du parcours utilisateur.
